# 🎲 Dice Rolling Game



A simple Python game where the user can roll a dice and get a random number between 1 and 6.




## Features

- Roll a dice using Python
- Random number generation (1–6)
- Replay option
- Input validation for "yes" or "no"





## 🕹️ How to Play

1. Run the Python file
2. Type "yes" to roll the dice
3. Type "no" to exit
4. Choose to play again or quit

python dice_game.py
